# ConditionExpansion
Дополнение для PlaceholderAPI для проверки условий.

### Формат
`%cond_{значение1_ОПЕРАТОР_значение2}?{РЕЗУЛЬТАТ_ЕСЛИ_ИСТИНА}:{РЕЗУЛЬТАТ_ЕСЛИ_ЛОЖЬ}%`

Чтобы вставить внутренний плейсхолдер из PlaceholderAPI, используйте символ `^` вместо `%`.
Пример: `^player_name^` вместо `%player_name%`.

### Операторы
- `==` — Равно
- `!=` — Не равно
- `>` — Больше
- `<` — Меньше
- `>=` — Больше или равно
- `<=` — Меньше или равно
- `==!` — Равно (с игнорированием регистра)
- `!==!` — Не равно (с игнорированием регистра)

### Примеры

1. Проверка ника (строковое сравнение):
   Если ник игрока Notch, вернет "Создатель", иначе - "Игрок".
   `%cond_{^player_name^==Notch}?{Создатель}:{Игрок}%`
